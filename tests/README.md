# Estrategia de pruebas

Las pruebas acompañarán cada módulo desde F3.

- Unitarias para cálculos y reglas.
- Integración para persistencia, periodos y exportaciones.
- Flujo para captura, corrección y anulación de movimientos.
- Regresión para calendario, compromisos y saldos negativos.
- Rendimiento con historiales amplios y dispositivos de recursos limitados.
- Recuperación ante interrupciones y datos incompletos.
